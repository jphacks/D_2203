## Usage

タスクは未着手，対応中，完了に分離されます．状況に応じてドラッグして変更してください

![task画像1](/assets/task1.png)

### 新規作成

新規作成するには新規ボタンを押して作成します

![task画像2](/assets/task2.png)

### プロパティ

必須のプロパティは次の通りです．かならず入力してください

|  プロパティ |  説明  | 
| ---- | ---- | 
|  タイトル  | タイトル |
|  ステータス  |  現在の進行状況  | 
|  ユーザー  | 対応者 | 
|  作成日 |  作成日(自動入力されます) |
